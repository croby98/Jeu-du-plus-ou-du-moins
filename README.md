## Getting Started <a name = "getting_started"></a>

You just need to have Rustup install on your machine.

For Linux/mac :

```
curl --proto '=https' --tlsv1.2 https://sh.rustup.rs -sSf | sh
```
For Windows :

```
https://www.rust-lang.org/tools/install
```

## Usage <a name = "usage"></a>
